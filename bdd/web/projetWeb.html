Initialisation

1. Construire la base de données Livraison dans MySQL:
  • en s’aidant de la version SQLite,
  • avec la modélisation en pj
  • en créant les tables à l’aide d’un creation.php; s’aider du TP Pokémon.
2. Créer un fichier remplissage.php pour insérer des données dans la base; utiliser les données de la base SQLite. Il est possible d’ajouter davantage de données.

Interface livreur

1.La page d’accueil se nommera: livreur.php:
  • Il s’agira d’un formulaire contenant une liste déroulante, qui permettra de choisir le nom du livreur.
  • La validation de ce formulaire dirigera vers le fichier gestion-livreur.php, en transmettant les informations nécessaires du livreur.
2. La page gestion-livreur.php permettra:
  • d’afficher les colis gérés par ce livreur et de modifier leur statut; l faudra créer un formulaire qui redirigera vers un fichier statut-livreur.php
  • de sélectionner des nouveaux colis à prendre en charge; il faudra créer un formulaire avec des cases à cocher qui redirigera vers un fichier selection-livreur.php

Interface client

1. la page d’accueil se nommera: client.php:
  • Il s’agira d’un formulaire contenant une liste déroulante, qui permettra de choisir le nom du client.
  • La validation de ce formulaire dirigera vers le fichier gestion-client.php, en transmettant les informations nécessaires du client.
2. La page gestion-client.php permettra:
  • d’afficher le statut des différents colis envoyés,
  • de déposer un nouveau colis; pour rappel, le dépôt d’un colis affecte un nouvel identifiant au paquet déposé dans la boîte aux lettres. Pour simplifier, on ne pourra envoyer un colis qu’à un autre client déjà présent dans la base.

Critères de notation

  • Nommer le dossier nom1-nom2-….zip, sans accent ni espace et le déposer dans le casier numérique.
  • Respect du cahier des charges précédents.
  • Tests qui seront effectués:
    - initialisation de la base
    - sélection d’un livreur
    - sélection de nouveaux colis à prendre en charge
    - changement de statut d’un colis
    - sélection d’un client
    - dépôt d’un colis
