Système de Gestion de Base de Données (SGBD)

  • Un logiciel

  • Un langage

  • Respect des contraintes
Information
Un Système de Gestion de Base de Données (SGBD) est un logiciel permettant de manipuler les données d’une base de données.
    Exemple:
  • MySQL
  • MariaDB
  • PostgreSQL
  • Oracle
  • SQLServer
  • SQLite

Edgar Codd propose le modèle relationnel.
Donald Chamberlin développe le langage SQL.
Le Structured Query Language permet d’interroger le SGBD.

Donald Chamberlin

    Le SGBD est construit pour respecter les contraintes d’intégrité:
  • Contrainte de domaine: type de données
  • Contrainte d’entité: clé primaire
  • Contrainte de référence: clé étrangère
